Site de cadastro de produtos de uma loja de doces <br>

Como usar: <br>


1 - Página de Login:<br>

◦ Clique em “Cadastrar” para criar um novo usuário.<br>
◦ O e-mail deve estar em um formato válido (ex: exemplo@gmail.com).<br>
◦ A senha deve conter no mínimo 8 caracteres, incluindo pelo menos um número.<br>

2 - Acesso ao Sistema:<br>

◦ Após o cadastro, é possível realizar login como usuário padrão para visualizar os produtos cadastrados.<br>
◦ Também é possível acessar como administrador, utilizando as credenciais:<br>
◦ Usuário: admin<br>
◦ Senha: admin123<br>

3 - Página do Administrador:<br>

◦ Permite cadastrar novos produtos, informando:<br>
◦ Nome do produto<br>
◦ Preço<br>
◦ URL da imagem<br>

4 - Página de Relatórios:<br>

◦ Exibe todos os produtos cadastrados.<br>
◦ Possibilita editar ou excluir itens conforme necessário.<br>
◦ Página de Vendas:<br>
◦ Mostra a lista de produtos disponíveis.<br>
◦ Permite alterar o status de cada produto entre “Disponível” e “Esgotado” — essa informação é refletida automaticamente na visualização do usuário padrão.<br>
